# 🌐 Mathematical Modeling Atlas

An interactive, beginner-friendly mathematical modeling knowledge platform designed for learning, competition preparation, and rapid model exploration.

👉 Live Website:  
https://gloriashang.github.io/Mathematical-modeling-atlas/

---

## 📌 Project Overview

This project builds a **comprehensive mathematical modeling knowledge atlas**, integrating problem types, model families, data workflows, and practical learning paths into a unified interactive system.

It aims to transform fragmented modeling knowledge into a **structured, navigable, and reusable learning ecosystem**.

---

## 🧠 Key Features

### 📚 1. Problem-to-Model Navigation
- Organizes modeling problems by type:
  - Prediction / Forecasting  
  - Optimization  
  - Evaluation / Decision-making  
  - Classification / Pattern recognition  
- Guides users from problem characteristics to suitable modeling approaches  

---

### 🧩 2. Model Library (Hierarchical)
- Covers major modeling families:
  - Optimization (LP, IP, DP, Heuristics)
  - Machine Learning (Regression, Classification, Clustering)
  - Statistical Models (Time Series, Hypothesis Testing)
  - Evaluation Models (AHP, TOPSIS, PCA)
  - Network & Graph Models  
- Structured as:
  - Category → Subcategory → Model → Explanation  

---

### 🛠️ 3. Data & Code Lab
- End-to-end workflow:
  - Data preprocessing  
  - Feature engineering  
  - Model implementation  
  - Result visualization  
- Integrated with reusable code templates  

---

### 📝 4. Paper Writing Studio
- Standard mathematical modeling paper structure:
  - Problem analysis  
  - Assumptions  
  - Model formulation  
  - Solution and results  
  - Discussion  
- Competition-oriented writing guidance  

---

### 🌍 5. Competition-Oriented Design
- Designed for:
  - CUMCM  
  - MCM/ICM  
  - Other modeling contests  
- Supports:
  - Fast model selection  
  - Strategy planning under time constraints  

---

### 🔍 6. Interactive Learning Experience
- Modular navigation system  
- Clickable knowledge nodes  
- Multi-level exploration  
- Beginner-friendly design  

---
